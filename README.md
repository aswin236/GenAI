# 🚀 CodeCraftHub

A simple REST API project built with **Python and Flask** that allows developers to track courses they want to learn.

This project is designed for beginners who are learning:
- REST API fundamentals
- CRUD operations
- JSON data handling
- HTTP methods
- Basic backend development

No database is used — all data is stored in a simple `courses.json` file.

---

# 📖 Project Overview

CodeCraftHub is a lightweight learning tracker API where developers can:

- Add new courses they want to learn
- Track progress
- Update course details
- Mark courses as completed
- Delete courses

It focuses purely on understanding how REST APIs work.

---

# ✨ Features

- ✅ Full CRUD operations (Create, Read, Update, Delete)
- ✅ JSON-based storage (No database required)
- ✅ Auto-generated ID
- ✅ Auto-generated timestamp (`created_at`)
- ✅ Input validation
- ✅ Proper HTTP status codes
- ✅ Error handling for invalid data
- ✅ Beginner-friendly structure
